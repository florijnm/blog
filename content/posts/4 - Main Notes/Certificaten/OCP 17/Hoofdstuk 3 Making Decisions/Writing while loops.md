Een while statement ziet er als volgt uit:

```
while(booleanExpression){
	//Body
}
```

Ook hier zijn de codeblock brackets niet verplicht bij *enkele* statements.

### Do/while
Je kunt ook een do/while loop gebruiken. Deze garandeert dat iig één keer de statement wordt uitgevoerd:

```
do{
	//Ik word altijd minimaal één keer uitgevoerd!
} while(booleanExpression);
```

### For loop
```
for(initialization; booleanExpression; updateStatement){
	//Body
}
```
